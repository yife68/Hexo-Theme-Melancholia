## 💻 安裝

### Git 安裝

在博客根目录里安装最新版【推荐】

```powershell
git clone -b main https://github.com/yife68/Hexo-Theme-Melancholia.git themes/Melancholia
```

暂不支持，因为我不会

## ⚙ 应用主题

修改hexo配置文件 `_config.yml` ，把主题改为 `Melancholia`

```
theme: Melancholia
```
